# Lattice

A 3D neural graph of 700 mental models. Each node is a decision-making framework — from Bayesian reasoning to game theory to behavioral economics. Edges represent semantic relationships between models. The graph fires like a brain.

Built with Next.js, Three.js, and Zustand.

## What it does

**Explorer** — Navigate a 3D graph of 700 mental models across 10 disciplines. Click nodes to explore their connections. Double-click for Synapse Mode — an immersive flythrough between connected models. Filter by discipline, spotlight edge types, search with Cmd+K.

**Oracle** — Describe a decision or situation in plain language. The Oracle (powered by Claude) maps it against all 700 models and returns a thinking framework: which models support your direction, which challenge it, and which shape the process. Each result includes a pointed question to ask yourself and a direct stance.

## The Graph

700 mental models spanning Probability, Investing, Behavioral Economics, Algorithms & ML, Economics, Financial Theory, Mathematics, Elementary Models, Philosophy, and Game Theory.

2,796 edges connecting them, classified into 6 semantic types:

| Type | Count | Meaning |
|------|-------|---------|
| Complementary | 809 | Strengthen each other when combined |
| Same Discipline | 695 | Related within the same field |
| Structural Kinship | 454 | Similar structure across different domains |
| Cross-discipline | 309 | Share concepts across fields |
| Prerequisite | 197 | One builds understanding for the other |
| Tensioning | 183 | Productive friction — opposing perspectives |
| Inversion | 59 | Direct opposites |

Edges were generated by embedding all 700 models with Voyage-3, computing cosine similarity, and classifying the top pairs with Claude.

## Getting Started

```bash
git clone https://github.com/philippbeer/lattice.git
cd lattice
npm install
npm run dev
```

Open http://localhost:3000.

### Oracle Setup

Oracle mode requires an Anthropic API key. You have two options:

**Option A: Bring your own key (recommended for public deployments)**
1. Click "Oracle" in the top navigation
2. Paste your API key in the inline key field
3. Your key is stored in your browser's localStorage only

**Option B: Server-side key**
```bash
cp .env.local.example .env.local
# Edit .env.local and add your Anthropic API key
```

## MCP Server

Lattice exposes its graph as an MCP (Model Context Protocol) server, so any compatible AI agent can query the mental models.

**Claude Desktop:**
```json
{
  "mcpServers": {
    "lattice": {
      "command": "npx",
      "args": ["@lattice/mcp-server"]
    }
  }
}
```

**Claude Code:**
```bash
claude mcp add lattice npx @lattice/mcp-server
```

7 tools available: `search_models`, `get_model`, `get_connections`, `list_disciplines`, `get_models_by_discipline`, `find_related`, `oracle`.

## Stack

- **Frontend**: Next.js 14, TypeScript, Tailwind
- **3D Rendering**: Three.js (InstancedMesh, custom GLSL shaders, post-processing)
- **State**: Zustand
- **Oracle**: Claude Sonnet via Anthropic API
- **Embeddings**: Voyage-3 (edge generation), sentence-transformers (gap filling)
- **Docs**: Nextra
